Description

colorFool is the ultimate color picking game. Learn different color modes while playing.

Decipher the code given in the magenta box below and choose the right square, to reveal a star. Progress through levels and gain score. When finished, Press the turqoise restart button to get a star rating, with a maxium of three. 3 levels of difficulty guarantee fun to everybody regardless of skill level.

Please contact me on Twitter with comments and suggestions. The bird icon is on the Options bar.

Interface

Keyboard shorcuts're given between square brackets when available.

  • CSS - Switches color mode to CSS named colors. [ s ]
  • HEX - Switches color mode to HEX colors. [ x ]
  • RGB - Switches color mode to RGB colors. [ b ]
  • HSL - Switches color mode to HSL colors. [ l ]
  • - Link to my Twitter.
  • - Toggles info pane. [ i ]
  • Restart - Restarts the game and ranks the last run. [ r ]
  • Easy - Switches to a 3 square board. [ e ]
  • Medium - Switches to a 6 square board. [ m ]
  • Hard - Switches to a 9 square board. [ h ]
  • The Board - Shows squares with different backgrounds. [1 - 9]
  • Level - Shows the current level.
  • The Color - Shows the right color code.
  • The Score - Shows the current score.
  • Inbox - Shows info, hints and jokes. Mostly jokes. Also shows your star rating after you restart.
